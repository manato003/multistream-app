# Lessons Learned

Claudeがユーザーから修正を受けた際に記録するパターン集。
セッション開始時に確認して同じミスを繰り返さない。

## 2026-02-23

### fflateインストール時にnode_modulesが破損した
- **状況**: `npm install fflate` 実行後にViteの依存関係が大量に消えてホワイトアウト
- **原因**: OneDrive上のプロジェクトでnpmがファイルロックを正しく処理できなかった
- **教訓**: 新しいnpmパッケージを追加する前にdevサーバーを停止する。OneDrive環境では特に注意。
- **対策**: パッケージ追加時は `Ctrl+C` でサーバー停止 → インストール → サーバー再起動の順で行う

### ドラッグ時に全フレームがリロードされるバグ
- **状況**: 枠の入れ替え時に全配信がリロードされる
- **原因**: StreamGridで `reloadKeys` をstateで管理し、StreamFrameの `key` に含めていたため並び替えで全フレームが再マウント
- **教訓**: `key` に含める値が変化すると対象コンポーネントが完全に再マウントされる。リロードキーはStreamFrame内部stateで管理する。
- **対策**: StreamFrameの `key={stream.id}` のみ。reloadKeyはStreamFrame内部で管理してplayerコンポーネントにだけ渡す。

### ドロップダウン外クリックで閉じる実装が配信画面に干渉
- **状況**: 設定ドロップダウンの「外クリックで閉じる」がiframe（配信画面）のクリックでも発火して再生が止まる
- **原因**: `mousedown` イベントリスナーをdocumentに貼っていた
- **教訓**: iframeの上にはmousedownイベントが伝播しないが、iframeの外でのクリックが意図せず配信に影響する場合がある
- **対策**: 外クリック閉じを削除。閉じる方法はマウスアウト or ハンバーガー再クリックのみ

### @handleがTwitchとして処理されるバグ
- **状況**: `@Popo_Ieiri` を入力すると `/embed/@Popo_Ieiri` になり再生エラー
- **原因**: `detectPlatformFromUrl` はURLにyoutube.com/youtu.beが含まれないと`null`を返す。`null`の場合はプラットフォームセレクターのデフォルト（Twitch）が使われるため、YouTube解決ロジックに到達しなかった
- **教訓**: `@xxx` 形式はURLではないがYouTubeチャンネルハンドルとして明示的に処理する必要がある
- **対策**: `detectPlatformFromUrl` に `@` で始まる入力をYouTubeとして扱うケースを追加

### React.memoへの書き換えで};と});が二重になるバグ
- **状況**: `const Foo = ({ ... }) => { ... };` を `const Foo = React.memo(({ ... }) => { ... });` に書き換える際、末尾の `};` を削除せず `});` を追加してしまい構文エラーになった
- **原因**: str_replaceで末尾だけを追加する形で編集したため、元の `};` が残った
- **教訓**: React.memoでラップする際は末尾の `};` を `});` に**置換**する。追加ではなく置換。
- **対策**: 編集後に必ずファイル全体を読み直して構文を目視確認する
