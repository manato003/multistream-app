# Tasks / 次回以降の実装候補

## 配信管理パネル（StreamSidePanel）改善

### 1. お気に入りフォルダへのチャンネル移動 UX
**現状の問題:** フォルダ間のドラッグ&ドロップが操作しにくい

**候補A: ドラッグハンドルの改善**
- 現在のドラッグハンドルをフォルダヘッダーへのドロップに対応させる
- フォルダ上にホバーしたときのビジュアルフィードバックを強化

**候補B: 編集モード**
- パネルに「編集」ボタンを追加、クリックで編集モード切替
- 編集モード時、各チャンネルにチェックボックスを表示
- 選択済みチャンネルをまとめて任意フォルダへ移動（ドロップダウン or フォルダをクリック）

### 2. 配信管理パネルのピン留め機能
- チャットパネルと同様のピン留めボタンを追加
- ピン留め時: パネルが常時表示、左側に固定（`app-main` に `padding-left` 追加）
- localStorage `streamPinned` に状態を保存
- App.tsx に `isStreamPinned` state 追加、`.app-main.stream-pinned` で padding 制御

### 3. パネル幅の可変対応
- フォルダツリーが深くなるにつれ横方向に広がることが想定される
- 固定幅（260px）→ コンテンツに合わせて可変
- 方針案: `min-width: 220px; width: max-content; max-width: 400px;`
- または、リサイズハンドルを追加してユーザーが任意に幅を調整できるようにする
